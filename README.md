**Tip Calculator & Bill Splitter**

A modern, responsive, and user-friendly web application designed for quick bill splitting and tip calculation. This app provides real-time updates as users input their bill amount, tip percentage, and the number of people.

**Key Features**

Real-time Calculation: Instant updates without any "Calculate" button.

Flexible Tip Options: Choose from preset buttons (10%, 15%, 20%) or enter a custom tip percentage.

Inline Validation: Immediate error feedback for invalid inputs (e.g., negative bill, zero people).

Responsive UI: Optimized for both mobile devices and desktops using modern CSS.

Clean Design: "Cyber-Luxe" inspired dark mode interface.

**Tech Stack**

HTML5: Structural markup.

CSS3: Styling, Flexbox/Grid layouts, and responsiveness.

JavaScript (ES6): Dynamic logic and event handling.
